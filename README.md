# DrinkStore 🍷

**DrinkStore** es una plataforma web desarrollada con un enfoque de diseño editorial y minimalista, creada para gestionar y presentar un catálogo de bebidas de forma elegante y funcional.

El proyecto destaca por su precisión técnica en el frontend, buscando una experiencia de usuario fluida y visualmente curada.

## 🚀 Características

- **Curaduría Visual:** Interfaz limpia que prioriza la imagen del producto.
- **Arquitectura Modular:** Estructura de archivos organizada para facilitar el mantenimiento.
- **Navegación Intuitiva:** Flujo de usuario simplificado y directo.

## 🛠️ Stack Tecnológico

Para este desarrollo se utilizaron las siguientes herramientas:

- **HTML5:** Marcado semántico para mejorar el SEO y la accesibilidad.
- **CSS3:** Estilizado avanzado enfocado en diseño minimalista.
- **JavaScript:** Interactividad y manejo de la lógica del catálogo.
